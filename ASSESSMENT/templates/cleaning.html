Missing Values Before Cleaning

The raw dataset uses ? for missing values throughout. Weight (97%), max_glu_serum (95%), and A1Cresult (83%) are the most affected columns. Columns with >30% missing that lack a clinical imputation rationale are dropped entirely.

Record Count at Each Cleaning Stage

Two main reductions: (1) removing {{ stats.dirty.deceased_patients }} deceased patients who cannot logically be readmitted, and (2) deduplicating to one encounter per patient (keeping the first) to prevent information leakage between train/test splits.

Readmission Class Distribution (After Cleaning)

The dataset has significant class imbalance: only ~{{ stats.readmission_rate }}% of patients are readmitted within 30 days (<30). This is addressed in the model by using balanced sample weights during training, so the classifier does not simply predict the majority class for every patient.

Before vs After — Column Transformations

Four examples of how dirty raw columns are transformed into model-ready features. Each panel shows the distribution before (red) and the resulting encoding after (blue).

  • Age: 10 string brackets ([60-70) etc.) → numeric midpoint (65). Red bars show raw bracket counts; blue histogram shows the final numeric distribution.
  • A1Cresult: string values plus 83% missing → ordinal 0–3, with "None" as a meaningful category (patient was not tested).
  • diag_1 (ICD-9): 700+ unique raw codes reduced to 9 clinically meaningful disease categories, eliminating data sparsity.
  • Insulin: string (No/Steady/Up/Down) → ordinal 0–3. The line overlay shows early readmission rate per category — "Down" (dose decreased) has the highest readmission risk.

Feature Engineering

Engineered Features

  • age_numeric — midpoint of age bracket ([60-70) → 65)
  • diag_1/2/3_cat_enc — 700+ ICD-9 codes → 9 categories (Diabetes, Circulatory, etc.)
  • num_meds_changed — count of medications with Up/Down dosage change
  • num_meds_used — count of medications currently being taken
  • discharge_grp — Home / Transfer+SNF / Home+Services / Other
  • admission_type_grp / admission_source_grp — grouped categorical admission context
  • total_prior_visits — outpatient + emergency + inpatient visits (prior year)
  • has_prior_inpatient — binary: any prior inpatient admission
  • high_risk_discharge — binary: discharge IDs with ≥25% readmission rate
  • insulin_down — binary: insulin dose was reduced at discharge
  • poor_glycaemic_control — binary: A1C ≥7% AND on insulin
  • long_stay / multimorbid — binary: ≥7 days / ≥7 diagnoses

Ordinal Encodings

  • a1c_result_enc: None=0, Normal=1, >7%=2, >8%=3
  • glu_serum_enc: None=0, Normal=1, >200=2, >300=3
  • insulin_enc: No=0, Steady=1, Up=2, Down=3
  • metformin_enc / glipizide_enc / glyburide_enc / glimepiride_enc: same scale
  • gender_enc: Female=0, Male=1
  • change_enc: No medication change=0, Changed=1
  • diabetes_med_enc: Not on diabetes medication=0, Yes=1
