Pipeline Stages

This application walks through the full data science pipeline:

  • Data Cleaning — Imputation, deduplication, standardisation of the raw dirty CSV.
  • Data Exploration — Pattern discovery with Matplotlib & Seaborn charts.
  • Predictive Model — Random Forest classifier for DIABETES / HEART DISEASE.

Dataset Overview

Columns: Patient_ID Age Gender Blood_Pressure Cholesterol BMI Smoker Diagnosis Admission_Date Notes

The raw dataset contained missing values, inconsistent casing, and Y/N vs YES/NO mismatches. These were resolved in the Data Cleaning stage using median, mean, and mode imputation strategies appropriate for each feature type.
