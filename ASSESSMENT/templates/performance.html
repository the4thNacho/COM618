GridSearchCV selected max_depth=None, min_samples_leaf=1 — the default, fully-grown settings. This means constraining tree depth did not improve CV ROC-AUC on this dataset. With only 52 samples, each 5-fold training split contains ~41 records, and the signal-to-noise ratio is too low for depth constraints to help.

Model Comparison (5-Fold CV)

Insight: The Random Forest (default / tuned) achieved the highest CV ROC-AUC ({{ stats.cv_roc_mean }}%). Logistic Regression and Naive Bayes scored below the random baseline on ROC-AUC, suggesting the decision boundary in this dataset is not well-captured by linear or independence-assuming models. SVM was near-random. This confirms RF is the appropriate choice — though all models suffer from the small labelled sample size.

Leave-One-Out CV vs Other Estimates

LOO CV trains on 51 samples and tests on 1, repeated 52 times. It uses the maximum possible training data, giving the least-biased estimate. LOO accuracy = {{ stats.loo_acc }}%. The divergence between the single hold-out split and the CV-based estimates illustrates how unreliable a single 13-sample test set is for evaluating performance.

Train vs Test Accuracy

100% train accuracy vs low test accuracy is memorisation, not generalisation. Even after GridSearchCV tuning, the default parameters were selected, confirming the overfitting is data-volume-driven rather than parameter-driven.

ROC Curve

The ROC curve plots TPR vs FPR. The dashed diagonal = random classifier (AUC=0.50). CV mean AUC = {{ stats.cv_roc_mean }}% > 50%, showing the model ranks positive cases better than chance on average across folds.

Per-Class Precision / Recall / F1

Precision = correct positive predictions / all positive predictions. Recall = correct positive predictions / all actual positives. F1 = harmonic mean. A class with 0% recall was never predicted on the test set.

Prediction Confidence Distribution

High confidence + low accuracy = a miscalibrated model. Mean confidence = {{ stats.mean_confidence }}% while test accuracy = {{ stats.test_acc }}% — the model is more confident than it is correct, a hallmark of overfitting.

5-Fold Cross-Validation Scores

CV accuracy mean = {{ stats.cv_acc_mean }}%, CV ROC-AUC mean = {{ stats.cv_roc_mean }}% (both above the 50% random baseline). High fold-to-fold variance reflects the small dataset — each fold only has ~10 test samples.
